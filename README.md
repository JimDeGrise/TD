# TD — To-Do приложение для Android

Простое приложение для управления задачами на Android, написанное на Kotlin с использованием Jetpack Compose, Room и Clean Architecture.

## Возможности

- **Создание задач** — добавьте задачу с названием и необязательным описанием.
- **Просмотр описания** — нажмите на задачу в списке, чтобы открыть диалог с полным описанием. Если описание не задано, отображается соответствующее сообщение.
- **Цветовая подсветка** — каждая задача выделяется одним из нейтральных пастельных цветов (зелёный, синий, лавандовый, персиковый, бирюзовый, розовый) для удобного визуального различия.
- **Удаление задач** — удалите задачу нажатием кнопки корзины.
- **Локальное хранение** — все задачи сохраняются в локальной базе данных Room и не теряются после перезапуска приложения.

## Технологии

| Компонент | Технология |
|-----------|-----------|
| UI | Jetpack Compose + Material 3 |
| Архитектура | MVVM + Clean Architecture (Data / Domain / UI) |
| База данных | Room 2.6.1 (KSP) |
| Реактивность | Kotlin StateFlow / Flow |
| Язык | Kotlin |

## Сборка и запуск

**Требования:** Android Studio Hedgehog или новее, JDK 17+, Android SDK 34.

```bash
# Клонировать репозиторий
git clone https://github.com/JimDeGrise/TD.git
cd TD

# Собрать debug-сборку
./gradlew assembleDebug

# Запустить unit-тесты
./gradlew test
```

Откройте проект в Android Studio и нажмите **Run ▶** для запуска на эмуляторе или устройстве.

## Использование

| Действие | Как |
|----------|-----|
| Добавить задачу | Нажать кнопку **＋** в правом нижнем углу |
| Просмотреть описание | Нажать на карточку задачи |
| Удалить задачу | Нажать иконку корзины на карточке задачи |
